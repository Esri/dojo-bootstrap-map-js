Слияние слоев

Этот инструмент копирует объекты из двух слоев в новый слой. Объединяемые слои должны содержать объекты одного типа (полигоны, линии или точки). Способом объединения и копирования полей можно управлять. Например:

  • У вас имеется три слоя Англии, Уэльса и Шотландии, а вы хотите создать один слой Великобритании.
  • У вас есть 12 слоев, каждый из которых содержит информацию о земельных участках в смежных тауншипах (участок площадью 36 кв. миль). Я хочу объединить их в один слой, сохранив только поля с одинаковыми названиями и типами из всех 12 слоев.

Если опция Использовать текущий экстент карты (Use current map extent) включена, объединяются только те объекты, которые отображаются в текущем экстенте. Если опция отключена, будут объединены все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Выбрать слой

Выберите слой для объединения со слоем анализа.

Обновить объединенные поля

По умолчанию, все поля обоих слоев переносятся в выходные данные. Вы можете управлять способом переноса полей, используя следующие опции:

  • Переименовать (Rename) – поле в выходных данных будет переименовано.
  • Удалить (Remove) – поле будет удалено из выходных данных.
  • Сопоставить (Match) – имена полей приведены в соответствие друг другу. Значения поля из слоя для объединения будут скопированы в соответствующие выходные поля.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью раскрывающегося поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
