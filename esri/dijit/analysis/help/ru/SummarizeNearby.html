Суммировать близлежащие (Summarize Nearby)

Этот инструмент находит объекты, которые расположены в пределах указанного расстояния от объектов на слое анализа. Расстояние может быть измерено по прямой, по времени в пути (например, в пределах 10 минут езды) или по расстоянию перемещения (в пределах 5 км). Затем вычисляется статистика для близлежащих объектов. Например:

  • Вычисляет численность населения в пределах пяти минут езды от предполагаемого местоположения нового магазина.
  • Вычисляет число съездов с автотрассы в пределах одной мили вдоль дороги от предполагаемого местоположения нового магазина для использования в качестве меры доступности.

Если опция Использовать текущий экстент карты (Use current map extent) включена, будут анализироваться только те объекты слоя анализа и слоя объединения, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты слоя анализа и слоя объединения, даже если они находятся вне текущего экстента карты.

Выберите слой для суммирования

Объекты этого слоя, находящиеся в пределах указанного расстояния от объектов в слое анализа, будут суммированы.

Найти ближайшие объекты, используя

Вы можете задать используемый тип измерения расстояния: расстояние по прямой, расстояние пути или время пути. Расстояние пути и время пути используют дорожную сеть и учитывают различные ограничения, такие как улицы с односторонним движением. Время в пути учитывает ваш выбор скорости движения: на основе текущего или типичного трафика. Текущие скорости определяются с помощью запросов к датчикам, типичные скорости вычисляются с помощью усреднения истории трафика.

Вы можете задать несколько расстояний. Объекты, находящиеся в пределах указанных расстояний (или на равным им) будут суммироваться.

Добавить статистику из

Дополнительно вы можете вычислить статистику любых числовых атрибутов ближайших точек.

При включении опции Число точек (Count of points) будет вычислено общее число соседних точек. Кроме того, вы можете вычислить статистику числовых атрибутов ближайших точек.

  • Сумма (Sum) – суммарное значение указанного поля
  • Среднее (Average) – вычисляет среднее арифметическое указанного поля
  • Минимум (Minimum) – определяет минимальное значение указанного поля
  • Максимум (Maximum) – определяет максимальное значение указанного поля
  • Стандартное отклонение (Std. Deviation) – стандартное отклонение для указанного поля

Выбрать поле для группировки

Это атрибут ближайших объектов, который может использоваться для вычисления статистики отдельно для каждого уникального значения атрибута. Например, предположим, что ваш слой анализа содержит свободные земельные участки, предназначенные для продажи, а ближайшими объектами являются контуры зданий. Одним из атрибутов контуров зданий является UseType, имеющий значения RETAIL, RESIDENTIAL, MANUFACTURING, PUBLIC и т.д. Чтобы охарактеризовать типы зданий вблизи каждого свободного участка, вы можете вычислить общую площадь для каждого уникального типа зданий в пределах двух километров от свободных участков, используя UseType как группировку по атрибуту. Статистика для каждой группы, наряду с количеством ближайших объектов, будет отображена во всплывающем окне результатов.

Добавить меньшинство, большинство

Эта опция активна, когда вы выбираете поле для группировки. Если вы хотите определить, какие атрибутивные значения в каждой группе слоя анализа составляют меньшинство или большинство в пределах указанного диапазона расстояния или времени, выберите Добавить меньшинство, большинство (Add minority, majority). Если вы указали опцию Добавить меньшинство, большинство (Add minority, majority), к результирующему слою будут добавлены два новых поля. Если вы также выберите Добавить процент (Add percentages), к результирующему слою будут добавлены еще два дополнительных поля, содержащих величины в процентах для большинства и меньшинства в пределах каждой группы.

  • Если вы выполняете суммирование соседних точек, меньшинство и большинство вычисляется на основе количества точек внутри группы в слое анализа в пределах указанного диапазона расстояния или времени.
  • Если вы выполняете суммирование соседних линий, меньшинство и большинство вычисляется на основе длин линий в пределах указанного диапазона расстояния или времени.
  • Если вы выполняете суммирование соседних областей, меньшинство и большинство вычисляется на основе площади в пределах указанного диапазона расстояния или времени.

Добавить проценты

Эта опция активна, когда вы выбираете поле для группировки. Выберите Добавить проценты (Add percentages) если вы хотите вычислить величину в процентах для каждого атрибутивного значения в пределах каждой группы. К результирующей таблице будет добавлено новое поле, содержащие величину в процентах для каждого атрибутивного значения в пределах каждой группы. Если вы также выберите Добавить меньшинство, большинство (Add minority, majority), к результирующему слою будут добавлены еще два дополнительных поля, содержащих величины в процентах для большинства и меньшинства в пределах каждой группы.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы (My Content) и добавлен на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью ниспадающего списка поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Content), в которую будет записан результат.
