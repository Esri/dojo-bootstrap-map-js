Обогатить слой

Этот инструмент обогащает ваши данные, собирая факты о людях, местах и деловой активности вокруг объектов ваших данных, что дает вам возможность получать дополнительную информацию о ваших местоположениях, которую вы не сможете получить, используя только карту. Например: что за люди живут здесь? Чем заняты люди, живущие в этом месте? Какой образ жизни они ведут, и чем они занимаются? Какие предприятии и организации имеются в этом месте?

В результате вы получите новый слой, содержащий всю демографическую и географическую информацию из полученных наборов данных. Эта новая информация добавляется в таблицу в виде полей.

Если опция Использовать текущий экстент карты (Use current map extent) включена, будут анализироваться только объекты слоя анализа и слоя площадей, которые отображаются в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты слоя анализа и слоя площадей, даже если они находятся вне текущего экстента карты.

Переменные обогащения

Щелкните Выбрать переменные (Select Variables), чтобы открыть Обзор данных (Data Browser). С помощью окна Обзор данных (Data Browser) вы сможете находить и выбирать отдельные переменные обогащения или сразу несколько переменных.

Район выбирается автоматически на основании ваших данных. Если в ваших данных содержится информация о других странах, то вы можете выбрать другой район.

Для облегчения доступа переменные обогащения организованы в наборы данных. Когда вы выберете набор данных, вы увидите только несколько наиболее популярных переменных обогащения этого набора. Если щелкнуть Листать (Keep Browsing), то будут показаны категории в данном наборе. В каждой из категорий вы можете либо выбрать целую подкатегорию, либо перейти в одну из подкатегорий для выборки отдельных переменных обогащения. Вы можете также использовать окно поиска для поиска определенной переменной в категории.

По мере выборки переменных для обогащения они будут добавляться в тележку для покупок, расположенную в углу браузера Обзор данных (Data Browser). Используйте тележку для покупок для наблюдения за количеством выбранных вами переменных.

Каждая выбранная вами переменная будет добавлена как поле в слое результатов.

Определите области для обогащения

Это области, которые следует обогатить дополнительной информацией, находящейся в подборке данных. Если ваш слой анализа содержит точки или линии, вокруг каждой точки или линии создаются области с расстоянием по прямой, расстоянием по пути или времени в пути.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью ниспадающего списка поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
