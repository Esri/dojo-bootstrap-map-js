Найти существующие местоположения

Этот инструмент выбирает в изучаемой области существующие объекты, соответствующие набору критериев. Эти критерии могут основываться на атрибутивных запросах (например, свободные земельные участки) и пространственных запросах (например, на расстоянии одной мили от реки).

Если опция Использовать текущий экстент карты (Use current map extent) включена, будут анализироваться только объекты слоя анализа, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут обрабатываться все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Найти объекты, которые удовлетворяют следующему выражению

Щелкните кнопку Добавить выражение (Add Expression), чтобы создать выражение. Вы можете добавить несколько выражений. После ввода выражения и щелчка на кнопке Запустить анализ (Run Analysis) объекты на слое анализа будут фильтроваться по каждому выражению. Те объекты, которые удовлетворяют всем выражениям, записываются в итоговый слой.

Существует два основных типа выражений – атрибутивные и пространственные. Атрибутивные выражения обращаются к значениям полей, например, где есть слово 'San Francisco'. Существует четыре типа пространственных выражений:

  • в пределах расстояния: объект должен быть в пределах заданного расстояния от объекта из другого слоя. Например, можно найти земельные участки, находящиеся не далее 500 метров от реки Лос-Анджелес.
  • полностью содержит: объект должен полностью содержать объект из другого слоя. Например, поиск водосборной области, в которой целиком помещаются земельные участки, предназначенные для промышленного использования. Ни одна часть участка не может быть вне водосборной области.
  • полностью внутри: объект должен быть полностью окружен объектом из другого слоя. Например, можно найти школьные округа, которые целиком входят в границы города. Ни одна часть школьного округа не может располагаться вне границ города.
  • пересекает: объект содержит часть объекта из другого слоя. Например, можно найти земельные участки, пересекающиеся с реками. Любая часть реки, которая проходит по участку или граничит с участком, пересекается с ним.

Каждое пространственное выражение имеет соответствующее отрицание; вне пределов расстояния, не полностью содержится в, не полностью внутри и не пересекает.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию зависит от типа анализа и имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью ниспадающего списка поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
