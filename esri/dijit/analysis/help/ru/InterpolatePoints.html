Интерполировать точки

Инструмент Интерполировать точки (Interpolate Points) позволяет прогнозировать значения в новых местоположениях на основе полученных измерений из набора точек. Этот инструмент обрабатывает значения точечных данных в каждой точке и возвращает площади, отсортированные по прогнозным значениям. Например:

  • В районе управления качеством воздуха имеются сенсоры, измеряющие уровни загрязнений. Инструмент Интерполировать точки (Interpolate Points) может использоваться для прогнозирования уровней загрязнения в местах, не имеющих таких сенсоров, но в которых находятся люди, относящиеся к группе риска, например, в школах или больницах.
  • Прогнозирование содержания тяжелых металлов в зерновых культурах на основе проб, взятых у отдельных растений.
  • Прогнозирование уровней питательных веществ в почве (азот, фосфор, калий и др.) и другие индикаторы (такие, как электропроводимость) с целью изучения их влияния на урожайность зерновых и расчета точного объема удобрений для отдельных участков поля.
  • В метеорологии инструмент используется для прогнозирования температур, осадков и связанных с ними переменных (например, кислотных дождей).

Выберите поле для интерполяции

Выберите поле, содержащее значения, которые вы хотите интерполировать.

Оптимизировать для

Выберите предпочтительные для вас скорость и точность. Для более точных прогнозов требуется больше времени.

Ошибки выходных прогнозов

Если опция выбрана, то будет создан слой стандартных ошибок для прогнозов, основанных на интерполяции. Именем выходного слоя будет имя слоя результатов с добавленными к нему ошибками.

Польза стандартных ошибок состоит в том, что они дают информацию о надежности значений прогноза. Существует простое правило, гласящее, что истинное значение в 95 процентах случаев будет находиться между двух стандартных ошибок прогнозного значения интерполяции. В качестве примера предположим, что прогнозное значение для нового местоположения равно 50, а стандартная ошибка – 5. Это означает, что по прогнозу этого инструмента истинное значение в данном местоположении равно 50, но оно также может быть равным и 40, и 60. Для расчета диапазона разумных значений надо умножить стандартную ошибку на 2, а затем добавить полученное значение к прогнозному для получения верхней границы диапазона или вычесть его из прогнозного значения, чтобы получить нижний предел диапазона.

Классифицировать по

Выберите, как будут группироваться прогнозные значения по областям.

  • Геометрический интервал – области создаются с учетом интервалов классов, имеющих геометрическую последовательность. Это позволяет добиться того, что в каждом диапазоне классов будет находиться примерно равное количество значений, и величина изменений по интервалам будет постоянной.
  • Равные интервалы – области создаются таким образом, что диапазон прогнозных значений одинаков для каждой области.
  • Равная область – области создаются таким образом, что число значений данных одинаково для каждой области. Например, если в данных высокие значения преобладают над низкими, то и областей для высоких значений будет создано больше.
  • Вручную – эта опция позволяет задать собственный диапазон значений для областей. Эти значения вводятся в параметр Граничные значения класса (Class break values).

Число классов

Задает число классов (диапазон прогнозируемых значений) в слое результатов. Каждый класс определяет границы областей. Значения этих классов определяются параметром Распределить по (Classify by), упомянутым выше.

Граничные значения класса

Введите требуемые граничные значения для классов, разделяя их пробелами. Например, если вы хотите разбить классы на значения 1,5, 3 и 7, введите 1.5 3 7. Эти значения определяют верхнюю границу значения для каждого класса, то есть количество введенных значений должно соответствовать количеству классов. Для местоположений, значения в которых будут превышать самое верхнее граничное значение, области созданы не будут. Необходимо ввести минимум два значения, но не более 32.

Вырезать результат по

Укажите слой с областью, для которой необходимо получить результат. Например, при интерполяции измерений загрязнений озера можно указать контур озера, и тогда результат будет представлен в пределах границ этого озера.

Интерполяция в этих местоположениях

Укажите слой с точечными местоположениями, для которых необходимо получить значения прогноза. Это позволяет делать прогнозы в определенных местах, представляющих интерес. Например, если слой анализа отображает показатели уровней загрязнения, то вы можете использовать этот параметр для прогнозирования уровней загрязнения в местах с людьми, относящимися к группам риска, например, в школах или больницах. Затем вы можете использовать эту информацию для выработки рекомендаций для местных органов здравоохранения.

Данный инструмент создаст точечный слой прогнозов в заданных местоположениях. Именем выходного слоя будет имя слоя результатов, к которому будет добавлено "Points".

Имя слоя результата

Укажите имя слоя, который будет создан в разделе Мои ресурсы (My Content) и добавлен на карту.

С помощью ниспадающего списка поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Content), в которую будет записан результат.
