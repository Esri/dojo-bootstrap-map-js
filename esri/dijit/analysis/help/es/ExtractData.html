Extraer datos

Con esta herramienta, puedes seleccionar y descargar los datos de un área de interés concreta. Las capas que selecciones se agregarán a un archivo ZIP, CSV o KMZ.

Capas para extraer

Estas son las capas de la tabla de contenido de las que puedes extraer entidades. Las capas que se marcan se extraen y se incluyen en el archivo ZIP o CSV resultante.

Para que una capa aparezca en la lista de capas para extraer, hay que ser el propietario de los datos o el propietario de los mismos debe habilitar la exportación de los datos. Para habilitar la exportación de datos de tu propiedad:

  • Abre la descripción del elemento de los datos.
  • Haz clic en EDITAR para modificar la descripción del elemento.
  • Desplázate hacia abajo hasta la sección Propiedades.
  • Activa Exportar datos.

Área que define las entidades que se pueden incluir en el archivo ZIP o CSV de salida. Puedes elegir entre las opciones siguientes:

  • La extensión visible actual del mapa ( Igual que la visualización).
  • Una de las capas de la tabla de contenido.
  • Dibuja un área para definir el área del estudio.
Se extraerán todas las entidades que estén totalmente dentro de la extensión del mapa. Para las entidades que cruzan el límite de la extensión del mapa o se intersecan con él, pero no están completamente contenidas en la extensión, tienes dos opciones:
  • Seleccionar entidades: se extraerán las entidades completas que cruzan el límite del área de estudio.
  • Recortar entidades: solo se extraerán las partes de las entidades que estén contenidas en el área de estudio.

Formato de datos de salida

Formato de los datos que se extraerán. Al exportar entidades de línea o de área a archivos CSV, solo se extraen atributos no espaciales. Al exportar entidades de puntos, se extraen tanto los atributos espaciales como los no espaciales.

Nombre de archivo de salida

Este es el nombre del elemento que se agregará a Mi contenido. El nombre del elemento se basa en el formato de datos de salida. Si el elemento ya existe, se te pedirá que confirmes si deseas sobrescribirlo.

En el cuadro desplegable Guardar resultado en, puedes especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
