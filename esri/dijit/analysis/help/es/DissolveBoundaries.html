Disolver límites

Las áreas que se superponen o comparten un límite se fusionan para formar una sola área.

Puedes controlar qué límites están fusionados especificando un campo. Por ejemplo, si tienes una capa de condados y cada condado tiene un atributo State_Name, puedes disolver los límites utilizando el atributo State_Name. Los condados adyacentes se fusionarán si tienen el mismo valor de State_Name. El resultado final es una capa de límites de estado.

Seleccionar método de disolución

Puede elegir entre dos métodos para fusionar las entidades de áreas:

  • Áreas que se superponen o son adyacentes: cuando los límites de dos o más áreas se superponen, estas áreas se fusionarán en una sola área. Las áreas adyacentes (es decir, las que comparten un límite) también se fusionarán en una sola área.
  • Áreas con el mismo valor de campo: las áreas que se superponen o comparten un límite se fusionarán en una sola área si tienen el mismo valor de campo.

    Por ejemplo, si tienes una capa de condados y cada condado tiene un campo State_Name, puedes disolver los límites utilizando el campo State_Name. Los condados adyacentes se fusionarán si tienen el mismo valor de State_Name. El resultado final es una capa de límites de estado.

Si se ha activado Utilizar la extensión de mapa actual, solo se disolverán las entidades de la capa de análisis que están visibles en la extensión de mapa actual. Si no se ha activado, se disolverán todas las entidades de la capa de análisis, aunque estén fuera de la extensión de mapa actual.

Agregar estadísticas

Cuando se fusionan las áreas, puedes resumir sus atributos. Por ejemplo, si estás disolviendo un límite entre condados en función del campo State_Name y cada condado tiene un campo Population, puedes calcular la suma de Population correspondiente al estado.

Los tipos de estadísticas disponibles son:

  • Suma: añade el valor total del campo especificado.
  • Valor medio: calcula el valor medio o media del campo especificado.
  • Mínimo: busca el valor más bajo de todos los registros del campo especificado.
  • Máximo: busca el valor más alto de todos los registros del campo especificado.
  • Desviación estándar: busca la desviación estándar de los valores del campo especificado.

Nombre de capa de resultados

Este es el nombre de la capa que se creará en Mi contenido y se agregará al mapa. El nombre predeterminado se basa en el tipo de capa de análisis. Si la capa ya existe, se te pedirá que confirmes si deseas sobrescribirla.

En el cuadro desplegable Guardar resultado en, puedes especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
