Enriquecer Camada

Esta ferramenta enriquece seus dados ao obter fatos sobre pessoas, lugares e negócios que cercam seus locais de dados e permite a você responder novas perguntas sobre locais que você não pode responder somente com mapas. Por exemplo: Que tipo de pessoa mora aqui? O que as pessoas gostam de fazer nesta área? Quais são seus hábitos e estilos de vida? Quais tipos de negócios existem nesta área?

O resultado será uma nova camada contendo todas as informações demográficas e geográficas das coletas de dados fornecidas. Estas novas informações são adicionadas como campos na tabela.

Se Utilizar extensão de mapa atual estiver selecionada, somente as feições na camada da área e análise visíveis dentro da extensão de mapa atual serão analisadas. Se não estiver selecionada, todas as feições em ambas a camada de análise e a camada de área serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Variáveis de enriquecimento

Clique em Selecionar Variáveis para abrir o Navegador de Dados. O Navegador de Dados o ajuda a localizar e selecionar variáveis de enriquecimento individualmente ou em coleções.

Uma região é automaticamente selecionada baseada em seus dados. Você pode escolher uma região diferente se os seus dados cobrirem mais de um país.

As variáveis de enriquecimento são organizadas em coleções de dados para acesso fácil. Quando você seleciona uma coleção de dados, o navegador mostra a algumas das variáveis de enriquecimento mais populares nesta coleção. Embaixo de Continuar Procurando, ele também mostra a categorias dentro da coleção. Dentro de cada categoria você pode selecionar subcategorias inteiras ou procurar em uma subcategoria e selecionar variáveis de enriquecimento individuais. Você também pode utilizar a caixa de pesquisa para localizar uma variável específica dentro de uma categoria.

Conforme você seleciona variáveis para incluir no enriquecimento, elas são adicionadas no carrinho de compras localizado no canto do Navegador de Dados. Utilize este carrinho de compras para acompanhar o número de variáveis que você selecionou.

Cada variável selecionada será adicionada como um campo na sua camada resultante.

Definir áreas para enriquecer

Estas são as áreas para enriquecer com informações adicionais encontradas na coleta de dados. Se a sua camada de análise tiver pontos ou linhas, as áreas serão criadas ao redor de cada ponto ou linha utilizando uma distância de linha reta ou um tempo do percurso.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdoe adicionada no mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, você será avisado para confirmar se deseja substituí-la.

Utilizando a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
