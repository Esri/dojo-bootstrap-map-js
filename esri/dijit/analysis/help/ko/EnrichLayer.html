레이어 보강

이 도구는 데이터 위치 주변의 인구, 장소 및 비즈니스에 대한 사실을 확인하여 데이터를 보강하며, 맵만으로는 확인할 수 없는 위치에 대한 새로운 질문의 답을 찾을 수 있도록 해줍니다. 예를 들면 다음과 같습니다. 여기에 어떤 사람들이 살고 있습니까? 이 영역의 사람들이 즐기는 일은 무엇입니까? 이들의 취미와 라이프스타일은 어떻습니까? 이 영역에는 어떤 종류의 비즈니스가 있습니까?

이 도구에서는 지정된 데이터 컬렉션으로부터 모든 인구 통계 및 지리 정보가 포함된 새 레이어가 결과로 생성됩니다. 이 새 정보는 테이블의 필드로 추가됩니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 분석 레이어와 영역 레이어의 피처만 분석되고, 이 옵션의 선택을 취소한 경우에는 현재 맵 범위 외부에 있는 피처를 포함하여 분석 레이어와 영역 레이어에 있는 모든 피처가 분석됩니다.

보강 변수

변수 선택을 클릭하여 데이터 브라우저를 엽니다. 데이터 브라우저를 통해 개별적 또는 통합적으로 보강 변수를 찾고 선택할 수 있습니다.

지역은 데이터를 기반으로 자동 선택됩니다. 데이터에 국가가 두 개 이상 포함된 경우 다른 지역을 선택할 수 있습니다.

보강 변수는 쉽게 접근할 수 있도록 데이터 컬렉션으로 구성되어 있습니다. 데이터 컬렉션을 선택하면 브라우저에는 해당 컬렉션에서 가장 인기 있는 보강 변수 일부가 표시됩니다. 계속 찾아보기에서도 컬렉션 내의 범주를 표시합니다. 각 범주 내에서 전체 하위 범주를 선택하거나 하위 범주로 드릴다운하고 개별 보강 변수를 선택할 수 있습니다. 또한 검색 상자를 사용하여 범주 내에서 특정 변수를 찾을 수도 있습니다.

보강에 포함할 변수를 선택하면 데이터 브라우저의 모서리에 있는 장바구니에 추가됩니다. 이 장바구니를 사용하여 선택한 변수의 개수를 추적합니다.

선택한 각 변수는 결과 레이어에 필드로 추가됩니다.

보강할 영역 정의

데이터 컬렉션에서 확인된 추가 정보로 보강할 영역입니다. 분석 레이어에 포인트나 라인이 포함된 경우 직선 거리 또는 운전 시간을 사용하여 각 포인트 또는 라인 주변의 영역이 생성됩니다.

결과 레이어 이름

내 컨텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 분석 레이어 이름을 기반으로 합니다. 레이어가 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
