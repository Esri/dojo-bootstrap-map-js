레이어 병합

이 도구는 두 레이어의 피처를 새 레이어에 복사합니다.병합할 모든 레이어에 같은 피처 유형(포인트, 라인 또는 영역)이 있어야 합니다.입력 레이어의 필드를 결합하고 복사할 방법을 제어할 수 있습니다.예를 들어

  • 잉글랜드, 웨일즈 및 스코틀랜드에 대한 3개의 레이어가 있는 경우 영국이라는 단일 레이어를 생성합니다.
  • 12개의 레이어가 있고 각 레이어에 인접 마을에 대한 필지 정보가 포함되어 있습니다.이러한 레이어를 단일 레이어로 결합하여 12개의 입력 레이어에서 이름과 유형이 같은 필드만 유지할 수 있습니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 피처만 병합됩니다.이 옵션의 선택을 취소한 경우 현재 맵 범위 외부에 있는 피처를 포함하여 두 레이어에 있는 모든 피처가 병합됩니다.

레이어 선택

분석 레이어와 병합할 레이어를 선택합니다.

병합 필드 수정

기본적으로 두 입력의 모든 필드가 출력으로 전달됩니다.필드가 출력으로 전달되는 방식을 다음과 같이 변경할 수 있습니다.

  • 이름 바꾸기 - 출력에서 필드의 이름이 바뀝니다.
  • 제거 - 출력에서 필드가 제거됩니다.
  • 일치 - 필드 이름이 동일하게 설정되고,병합할 레이어의 필드 값이 일치하는 출력 필드에 복사됩니다.

결과 레이어 이름

내 컨텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다.기본 이름은 분석 레이어 이름을 기반으로 합니다.레이어가 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
