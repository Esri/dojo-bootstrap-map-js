Daten extrahieren

Mit diesem Werkzeug können Sie Daten für einen bestimmten Interessenbereich auswählen und herunterladen. Ausgewählte Layer werden einer ZIP-,r CSV- oder KMZ-Datei hinzugefügt.

Zu extrahierende Layer

Dies sind die Layer im Inhaltsverzeichnis, aus dem Features extrahiert werden können. Aktivierte Layer werden extrahiert und in die resultierende ZIP- oder CSV-Datei eingefügt.

Damit der Layer in der Liste der zu extrahierenden Layer angezeigt wird, müssen Sie entweder der Besitzer der Daten sein oder der Besitzer der Daten muss das Exportieren der Daten aktivieren. So aktivieren Sie den Datenexport für Daten, die Sie besitzen:

  • Öffnen Sie die Elementbeschreibung der Daten.
  • Klicken Sie auf BEARBEITEN, um die Elementbeschreibung zu bearbeiten.
  • Führen Sie einen Bildlauf zum Abschnitt Eigenschaften durch.
  • Aktivieren Sie Daten exportieren.

Kartenausschnitt

Die Fläche zur Definition welche Features in die Ausgabe-ZIP- bzw. die CSV-Datei eingefügt werden. Ihnen steht Folgendes zur Auswahl:

  • Die aktuell sichtbare Ausdehnung der Karte ( Gleich wie Anzeige).
  • Einer der Layer in Ihrem Inhaltsverzeichnis.
  • Zeichnen einer Fläche zum Definieren des Interessensbereichs.
Alle Features, die sich vollständig innerhalb der Kartenausdehnung befinden, werden extrahiert. Für Features, die die Grenze der Kartenausdehnung kreuzen oder überschneiden, sich jedoch nicht vollständig innerhalb der Ausdehnung befinden, sind zwei Optionen verfügbar:
  • Features auswählen – Ganze Features, die die Grenze des Untersuchungsgebiets kreuzen, werden extrahiert.
  • Features ausschneiden – Nur die Teile des Features, die sich innerhalb des Untersuchungsgebietes befinden, werden extrahiert.

Format der Ausgabedaten

Das Format der Daten, die extrahiert werden. Beim Exportieren von Linien- oder Flächen-Features in CSV-Dateien werden nur nicht räumliche Attribute extrahiert. Beim Exportieren von Punkt-Features werden sowohl räumliche als auch nicht räumliche Attribute extrahiert.

Name der Ausgabedatei

Dies ist der Name des Elements, das zu Eigene Inhalte hinzugefügt wird. Der Elementname basiert auf dem ausgewählten Format der Ausgabedaten. Wenn das Element bereits vorhanden ist, müssen Sie angeben, ob es überschrieben werden soll.

Über das Dropdown-Feld Ergebnis speichern in können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert werden soll.
