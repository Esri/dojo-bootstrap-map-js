Dissolver Limites

Áreas que se sobrepõem ou partilham um limite são fundidas de forma a formar uma única área.

Pode controlar quais os limites a serem fundidos, especificando um campo. Por exemplo, se tem uma camada de municípios, e cada município tem um State_Name atributo, pode dissolver os limites utilizando o State_Name atributo. Se os municípios adjacentes tiverem o mesmo valor serão fundidos State_Name. O resultado final é uma camada com todos os limites fundidos.

Escolher o método para dissolver

Pode escolher um dos dois métodos para fundir a área dos elementos:

  • Áreas que se sobrepõem ou são adjacentes-Duas ou mais áreas cujos limites se sobrepõem serão fundidos numa área. Além disso, as áreas que são adjacentes (isto é, que partilham um limite) serão fundidos apenas numa área.
  • Áreas com o mesmo valor do campo-As áreas que se sobrepõem ou partilham um limite serão fundidos apenas numa área, se tiverem o mesmo valor do campo.

    Por exemplo, se tiver uma camada de municípios, e cada município tiver um State_Name campo, pode dissolver os limites utilizando o State_Name campo. Se os municípios adjacentes tiverem o mesmo valor, serão fundidos State_Name. O resultado final é uma camada com todos os limites fundidos.

Se a opção Utilizar a extensão atual do mapa estiver ativa, apenas os elementos que estão visivéis na atual extensão do mapa serão dissolvidos. Se não estiver ativa, todos os elementos da camada de entrada serão dissolvidos, mesmo que estejam fora da atual extensão do mapa.

Adicionar estatística

Quando as áreas são fundidas, é possível resumir os respectivos atributos. Por exemplo, se estiver a dissolver os limites entre municípios State_Name,e cada um tiver um Population campo, pode calcular a soma Population para todos os municipios.

Tipos de estatística disponíveis:

  • Soma-Adiciona o valor total para o campo especificado
  • Média-Calcula a média para o campo especificado
  • Mínimo-Encontra o menor valor para todos os registos do campo especificado
  • Máximo-Encontra o maior valor para todos os registos do campo especificado
  • Desvio -padrão-Encontra o desvio-padrão dos valores do campo especificado

Nome da camada de resultados

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome por defeito é baseado no nome da camada de análise. Se a camada já existir, será questionado se pretende substituir a camada existente.

Ao utilizar a opção Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
